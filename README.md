## Description

- Randomly generates cat sounds.
- 랜덤한 고양이 소리를 생성합니다.

## Installation

```shell
npm install random-cat-word
# or
yarn add random-cat-word
```

## Usage

```typescript
import {generate} from "random-cat-word/dist/cjs/main";

console.log(generate(10))
```
